ID Explained

==Summary== An OOMP ID at first glance can look like a random collection of letters and numbers (ie. '). However it is in fact lovingly crafted to contain enough detail about a part to make it unique, describe it well, and deduce it's name. ==ID== Each OOMP ID has 5 parts. ===.:Type:.=== This denotes the type of a component. (ex. ' - Capacitor (Tantalum), ' - Resistor) ===.:Size:.=== This denotes the major size of a component, usually used to define it's physical form factor. (ex. ' - 2.54 mm, ' - SMD 0603, ' - 1/4 Watt Resistor) ===.:Color:.=== This denotes the color of the component, or if relevant the material it is made from. (ex. ' - Red, ' - Clear, ' - Metal, ' - None) ===.:Description:.=== This denotes either an additional detail about a component (ex. ' - Frosted ' - Standard) or the value of the component. (ex. ' - 220 uf, ' - 36 Pole, ' - 560 Ohm) ===.:Index:.=== This is used to fine tune a component either if a particular supplier, or a special specification not catered to in the other elements is required. (ex. ' - +/-1% Precision, '1 - Normal) ' (ex. ' - 1/4 Watt 560 Ohm Resistor) ==Name== An OOMP ID can also be used to construct the name of a component. To do this we use the 5 elements of the ID and a conversion chart. Name: ' (ex. ' - 1/4 Watt 560 Ohm Resistor ' - 10mm Frosted Green LED) ==Hex ID== Each component is assigned a unique hexadecimal code. This is used to make linking to the component easy. (ex. ' 2.54mm 3 Pin Black Crimp Housing Hex ID: DA03 Link: http://oom.lt/DA03/)
